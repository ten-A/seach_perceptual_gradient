# seach_perceptual_gradient
Search Perceptual Gradient in Illustrator documents(Consept)

ここは「知覚的グラデーションを検出する手段を考えるリポジトリです。
実行ファイルは「searchPerceptualGradient.jsx」です。一応ファイルの保護はやっていますが、実行前に保存は行っておいて下さい。
ドキュメント定義上のグラデーションの数をカウントしていますので、実体が削除されて定義のみが生きているグラデーションもカウントします。しかしながら、過去に知覚的グラデーションを利用していたということでもありますので、一応確認を行っていただいたほうが良いでしょう。検出０のドキュメントは知覚的グラデーションが作られたことのないドキュメントなのでそのまま工程を進めても問題ないでしょう。

で、なにかアイデアのある方、是非issuesやPullRequestを投げ込んで下さい。
